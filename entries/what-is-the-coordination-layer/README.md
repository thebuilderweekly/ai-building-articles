# What is the coordination layer?

Building pillar, systems topic. Defines the coordination layer as the infrastructure between specialized agents that lets them share state, route work, and operate without a human copy-pasting context between windows. Covers the four components (shared state, event bus, identity and permissions, context protocol) and explains why MCP alone is not enough.
