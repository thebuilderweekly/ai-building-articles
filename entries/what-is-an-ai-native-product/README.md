# What is an AI-native product?

Foundation article in the Building pillar. Defines the distinction between AI-native products (whose core value cannot exist without AI) and AI-enhanced products (that bolt AI features onto existing value). Walks the spectrum from pure non-AI to AI-native with concrete named examples, explains why the distinction determines structural moats, and gives readers a two-question test for their own product.

Target length: 2000-2500 words. Pillar: building. Topic: product.
