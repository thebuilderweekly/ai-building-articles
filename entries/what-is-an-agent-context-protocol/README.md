# What is an agent context protocol?

Building pillar, protocols topic. Defines an agent context protocol as the structured contract that lets one agent or tool hand context to another agent without a human reformatting the data. Walks through MCP as the canonical example, the four properties of a real protocol (schema, identity, capability discovery, error model), emerging patterns beyond MCP, the protocol-vs-glue trade-off, and a five-question rubric for evaluating one before adopting. ~2,590 words.
